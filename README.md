Student Management System (Python)

This is a simple command-line program for managing student records. It uses a JSON file to store data so that records remain saved even after closing the program.

Features

Add new students

View all students

Search students by ID

Update student information

Delete student records

Data is stored in a JSON file called "students.json"

Requirements

Python 3.x

No external libraries are required
